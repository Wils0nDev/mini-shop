# MiniShop API

1. Clonar proyecto
2. ejecutar ```yarn install```
3. Clonar archivo env.template y renombrarlo a .env
4. Cambiar las variables de entorno.
5. Levantar la BD
```docker-compose up -d```
6. Levatar aplicación ```yarn start:dev ``` 
  * Si en caso sale error de conexión, "Crear bd".
